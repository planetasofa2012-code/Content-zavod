# 📋 Техническое задание: Контент-завод

> Формируется на основе идеи и исследования. Это скелет проекта.

---

## 1. Описание продукта

**Контент-завод** — комплексная AI-система для мебельного бизнеса, которая:
1. **Генерирует и публикует контент** на 7+ площадок автоматически
2. **Консультирует клиентов** через AI-агента на каждой площадке (до замера)
3. **Ведёт воронку продаж** по принципу «3 касания» (без продажи в лоб)
4. **Обучается** на диалогах с клиентами и интервью с владельцем
5. **Автоматизирует рутину**: сметы, отзывы, CRM, ресайкл контента

### Кто использует
**Александр Самосудов** — работает полностью один. Нет сотрудников. AI-система = виртуальный отдел маркетинга и продаж 24/7.

### Ключевая ценность
Александр тратит **1–2 минуты** (отправить фото + голосовое/текст) → получает готовые посты на 7 площадках + AI-агент общается с клиентами → клиенты сами записываются на замер.

---

## 2. Компоненты системы

### Компонент 1: Контент-завод (бот-генератор)
Telegram-бот для генерации и публикации контента на 7+ площадок.

### Компонент 2: AI-агент консультант
Чат-бот на каждой площадке: консультация → расчёт → запись на замер.

### Компонент 3: AI-интервьюер (генератор базы знаний)
Бот задаёт вопросы как клиент → Александр отвечает → ответы = база знаний + контент.

### Компонент 4: Лендинг
Сайт-визитка: портфолио, калькулятор, AI-чат, форма заявки.

## 5. Пользовательские сценарии

### Сценарий 1: Быстрый пост с фото (MVP)
```
1. Александр сфотографировал готовую кухню на объекте
2. Открыл Telegram → отправил фото боту
3. Написал: «Кухня в скандинавском стиле, МДФ эмаль, фурнитура Blum»
4. Бот через AI сгенерировал посты для каждой площадки:
   - Telegram: короткий + эмодзи + ссылка на лендинг
   - VK: подробный + вопрос для вовлечения
5. Бот показал предпросмотр
6. Александр нажал ✅ «Опубликовать»
7. Посты ушли в TG-канал и VK-группу за 5 секунд
```

### Сценарий 2: AI-агент ведёт клиента до замера (MVP)
```
1. Клиент увидел пост в VK с фото кухни
2. Написал в группу: «Добрый день, а сколько стоит такая кухня?»
3. AI-агент ответил:
   «Здравствуйте! 😊 Красивый выбор! Стоимость зависит от размеров
   и материалов. Подскажите — какой примерно размер кухни у вас?
   И что важнее — чтобы было бюджетно или чтобы выглядело премиально?»
4. Клиент: «3 метра, хочу чтобы красиво, бюджет 200-250»
5. AI: «Отлично! Для 3-метровой кухни в этом бюджете отлично подойдут
   фасады МДФ эмаль — это как раз на фото. По моему расчёту,
   примерно 180–220К в зависимости от фурнитуры.
   Хотите записаться на бесплатный замер? Наш мастер приедет,
   сделает точный расчёт и покажет 3D-эскиз — всё бесплатно 🏠»
6. Клиент: «Да, давайте»
7. AI: «Супер! Когда удобно? Могу предложить среду после 15:00
   или субботу утром. Напишите адрес — и я запишу вас»
8. → AI записал: дата, время, адрес, бриф
9. → Александру пришло уведомление в Telegram:
   «🔔 Новый замер: Пн 15:00, ул. Ленина 5, кухня 3м, бюджет 200-250К»
```

### Сценарий 3: Голосовой пост (V2)
```
1. Александр на замере, руки заняты
2. Записал голосовое: «Сегодня сдали шкаф-купе, клиентка
   в Ленинском районе, зеркальные двери, полки под обувь,
   очень довольна, три года мечтала»
3. Бот расшифровал (Whisper) → сгенерировал пост-кейс
4. Александр глянул, нажал ✅
5. Опубликовано на 7 площадок
```

### Сценарий 4: Калькулятор на лендинге (V2)
```
1. Клиент нашёл лендинг через Pinterest (пин с фото кухни)
2. Открыл калькулятор на сайте
3. Выбрал: Кухня → 3.5м → МДФ эмаль → фурнитура Blum
4. Получил: «≈ 190 000 – 250 000 ₽»
5. Нажал «Записаться на бесплатный замер»
6. AI-чат на сайте уточнил детали → записал на замер
```

### Сценарий 5: AI-интервьюер наполняет базу знаний (MVP)
```
1. Утром AI-бот прислал: «Александр, клиент спрашивает:
   "А чем МДФ эмаль лучше плёнки? Я не понимаю разницу"
   Как бы вы ответили?»
2. Александр записал голосовое (30 секунд)
3. Бот расшифровал → сохранил в базу знаний
4. Сгенерировал пост: «МДФ или плёнка? Разбираю за 30 секунд 👇»
5. Александр нажал ✅ → пост ушёл на 7 площадок
6. AI-менеджер теперь может сам ответить на этот вопрос клиенту
```

### Сценарий 6: Переписка → знания + контент (V2)
```
1. Александр переслал боту переписку с клиентом (скриншот или текст)
2. AI извлёк: 3 вопроса клиента, 3 ответа, 1 возражение
3. → Новые записи в базу знаний
4. → 2 поста для соцсетей (ответ на возражение + FAQ)
5. → AI-менеджер стал умнее на 3 вопроса
```

### Сценарий 7: Автосбор отзыва (V2)
```
1. В CRM заказ Ивановых перешёл в статус «Монтаж завершён»
2. Через 7 дней AI автоматически написал клиенту:
   «Здравствуйте! Как ваша новая кухня? Всё ли хорошо? 😊»
3. Клиент ответил: «Всё супер, спасибо!»
4. AI: «Буду благодарен за пару слов — это поможет другим людям
   выбрать надёжного мастера. Можете написать прямо сюда 🙏»
5. Получен отзыв → пост + лендинг + просьба на Яндекс.Карты
```
