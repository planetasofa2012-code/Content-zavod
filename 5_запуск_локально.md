## Шаг 1: Настройка .env

Скопируй `.env.example` в `.env` и заполни:

```bash
cp .env.example .env
```

**Минимум для MVP (только контент-бот + TG-публикация):**

```env
TG_CONTENT_BOT_TOKEN=   # токен от @BotFather
TG_AGENT_BOT_TOKEN=     # токен второго бота (для клиентов)
TG_CHANNEL_ID=          # ID канала, например -1001234567890
OWNER_ID=               # твой Telegram ID (узнать: @userinfobot)
OPENROUTER_API_KEY=     # ключ с openrouter.ai
```

**Как получить TG_CHANNEL_ID:**
1. Добавь бота в канал как администратора
2. Перешли любое сообщение из канала боту @userinfobot
3. Скопируй `Chat ID` (начинается с -100...)

## Шаг 4: Тест контент-бота

1. Открой Telegram → найди своего бота (по имени из @BotFather)
2. Отправь `/start` → должно прийти меню
3. Нажми «Создать пост»
4. Отправь фото любой мебели
5. Напиши описание: `«Кухня МДФ эмаль, скандинавский стиль»`
6. Выбери тип: «Портфолио»
7. Дождись генерации (5–15 сек)
8. Нажми ✅ «Опубликовать»
9. Проверь канал @kuhnya154 — пост должен появиться

---

## Шаг 5: Тест AI-агента

1. Открой второй бот (agent_bot)
2. Отправь `/start`
3. Напиши: `«Сколько стоит кухня 3 метра?»`
4. Проверь ответ — должен быть простым, человечным, без канцелярита
5. После 3 сообщений должна появиться кнопка «Записаться на замер»

## Возможные ошибки и решения

| Ошибка | Причина | Решение |
|--------|---------|---------|
| `Unauthorized` | Неверный токен бота | Проверь `TG_CONTENT_BOT_TOKEN` в `.env` |
| `Chat not found` | Неверный `TG_CHANNEL_ID` | Убедись, что бот — admin канала, ID с -100... |
| `OpenRouter error 401` | Неверный API-ключ | Проверь `OPENROUTER_API_KEY` |
| `OpenRouter error 402` | Закончился баланс | Пополни баланс на openrouter.ai |
| `ModuleNotFoundError` | Не установлены пакеты | `pip install -r requirements.txt` |
| Бот не отвечает | Фильтр OWNER_ID | Убедись, что `OWNER_ID` = твой реальный Telegram ID |
