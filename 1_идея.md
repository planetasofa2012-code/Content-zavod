### 🎙️ AI-интервьюер (генератор базы знаний + контента)

> **Проблема**: Александр работает один, времени нет. Выдумывать посты и описывать знания для AI — нереально.
>
> **Решение**: AI-бот задаёт вопросы **как клиент** → Александр просто отвечает (голосовым или текстом) → ответы становятся **базой знаний** и **контентом** одновременно.

#### Как работает

```
AI-интервьюер (бот в Telegram):
───────────────────────────────
🤖 «Александр, представьте: клиент спрашивает —
    "А чем МДФ эмаль лучше плёнки? Я не понимаю разницу"
    Как бы вы ответили?»

👨 Александр отвечает голосовым (или текстом):
   «Смотрите, плёнка — это как наклейка на материале...
   МДФ эмаль — это когда фасад красят как автомобиль,
   несколько слоёв... Плёнка дешевле, но через 5 лет
   может начать отходить. Эмаль дороже, зато на 15-20 лет...»

         ↓                    ↓                    ↓
    БАЗА ЗНАНИЙ          КОНТЕНТ               AI-ПРОМПТ
    ─────────────       ────────────          ──────────────
    FAQ: «МДФ vs        Пост: «МДФ или       AI-агент теперь
    плёнка» →           плёнка? Разбираю     знает и может
    Supabase            за 30 секунд 👇»     ответить клиенту
                        → на 7 площадок      так же, как
                                             Александр
```

#### Что это даёт (3 в 1)

| Результат | Описание | Без доп. усилий? |
|-----------|---------|-----------------|
| **1. База знаний** | Ответы Александра сохраняются в Supabase → AI-менеджер учится отвечать как Александр | ✅ Автоматически |
| **2. Контент для соцсетей** | Каждый ответ = готовый пост (AI оформляет и адаптирует) | ✅ Автоматически |
| **3. FAQ для лендинга** | Вопросы и ответы → раздел FAQ на сайте | ✅ Автоматически |

#### Типы вопросов AI-интервьюера

| Категория | Пример вопроса |
|-----------|---------------|
| **Материалы** | «А чем ЛДСП отличается от МДФ? Клиент не понимает» |
| **Цены** | «Клиент спрашивает: сколько стоит кухня 3 метра?» |
| **Процесс** | «Как проходит замер? Что мне нужно подготовить?» |
| **Страхи** | «А если мне не понравится результат? Есть гарантия?» |
| **Сроки** | «Сколько займёт производство кухни?» |
| **Фурнитура** | «Зачем платить за Blum, если есть дешевле?» |
| **Кейсы** | «Расскажите про свой последний заказ — что было, как сделали?» |
| **Конкуренты** | «Почему к вам, а не в магазин готовых кухонь?» |

#### Режимы работы

- **Режим «Интервью»** — AI задаёт 3–5 вопросов подряд (5–10 минут в день)
- **Режим «Случайный вопрос»** — 1 вопрос в день (утром, в пуш-уведомлении)
- **Режим «После замера»** — после каждого замера AI спрашивает: «Что было? Какая кухня? Что интересного?» → кейс-пост
- **Режим «Переписка»** — Александр пересылает реальную переписку с клиентом → AI извлекает: вопросы, ответы, возражения, итог → пополняет базу знаний + генерирует контент из живого диалога

#### Почему это решает главную боль

> Александр **не выдумывает** контент. Он просто **отвечает на вопросы**,
> которые ему и так задают клиенты каждый день.
> AI превращает эти ответы в посты, базу знаний и FAQ — автоматически.

        └────────────────────────┴────────────────────────┘

- 🔄 AI-менеджер **логирует все диалоги** с клиентами → анализирует: какие вопросы задают чаще? где клиент «отваливается»? какие ответы работают лучше?
- 📈 **Автоматическое улучшение**: новые вопросы клиентов → новые записи в базу знаний → AI отвечает точнее
- 🎯 **Новые темы для контента**: если 5 клиентов спросили «а почему так дорого?» → это тема для поста и FAQ

#### «Парадокс Барсика» (критично для AI-агента и контента!)

> **Проблема**: крутые специалисты создают контент для своего отражения в зеркале, а не для клиента. Клиент чувствует себя «Барсиком на защите докторской» — и уходит.
>
> **Вывод**: AI-агент и контент должны говорить **на языке клиента**, а не с трибуны.

Принципы:
- 🎯 Клиенту не нужен «меморандум в МКАС». Ему нужно чувствовать, что его проблему не посчитают ерундой
- 💬 AI-агент говорит как **друг, который разбирается в мебели**, а не как профессор кафедры деревообработки
- 🏠 Пример ответа AI: *«Здравствуйте! Расскажите, что хотите — может, кухню новую, шкаф или гардеробную? Я помогу разобраться с размерами и материалами, всё расскажу простым языком 😊»*
- ❌ Не так: *«Наша компания специализируется на производстве корпусной мебели из ЛДСП, МДФ с эмалевым покрытием, используя фурнитуру Blum/Hettich...»*

## Как должен работать контент-завод

### Идеальный сценарий (минимум усилий для владельца)

```
┌──────────────────┐     ┌──────────────────┐     ┌──────────────────┐
│  ВХОД            │     │  ОБРАБОТКА       │     │  ВЫХОД           │
│                  │     │                  │     │                  │
│ • Фото с объекта │────►│ • AI генерирует  │────►│ • Пост в Insta   │
│ • Голосовое      │     │   текст          │     │ • Пост в Telegram│
│   сообщение      │     │ • Подбирает      │     │ • Пост в VK      │
│ • Заметка        │     │   хештеги        │     │ • Пин в Pinterest│
│                  │     │ • Адаптирует под  │     │                  │
│                  │     │   площадку       │     │ • По расписанию  │
│                  │     │                  │     │ • или сразу      │
└──────────────────┘     └──────────────────┘     └──────────────────┘
         │                                                │
    1–2 минуты                                     Автоматически
   усилий владельца                               на 4 площадки
```

### Минимальное участие владельца
1. **Сфотографировать** готовую работу на телефон
2. **Отправить** фото боту (в Telegram) + голосовое/текст с описанием
3. **Подтвердить** сгенерированный пост (или отредактировать)
4. Бот **сам публикует** на все площадки

### Автопилот (контент-план)
- AI генерирует **контент-план на неделю/месяц**
- Экспертные посты, советы, вовлечение — без фото, на основе знаний владельца
- Публикация по расписанию
- Владелец только утверждает или корректирует

## Направления автоматизации (полная картина)

### 1. Контент-завод (SMM) ← ПРИОРИТЕТ
- Автоматическая генерация контента для социальных сетей
- Планирование и публикация постов
- Ведение 7+ площадок одновременно

### 2. AI-менеджер-консультант
- Бот-консультант на каждой площадке (свой стек, без Suvvy)
- Консультация → расчёт → запись на замер
- Самообучение на диалогах с клиентами

### 3. AI-интервьюер
- Задаёт вопросы как клиент → Александр отвечает → база знаний + контент
- Режимы: интервью, случайный вопрос, после замера, переписка

### 4. Лендинг + калькулятор
- 11 блоков продающей страницы
- AI-чат виджет, калькулятор заказов, портфолио

### 5. 📸 AI-фотограф (подсказки по съёмке)
- Чек-лист из 5 кадров: общий вид, деталь, наполнение, «до», с хозяином
- Подсказки прямо в Telegram: «Отлично! Теперь сделай крупный план фурнитуры 👆»
- **Результат**: даже с телефона фото выглядят профессионально

### 6. 🔔 Автосбор отзывов
- Через 7 дней после монтажа AI сам пишет клиенту с просьбой об отзыве
- Отзыв → пост + лендинг + просьба на Яндекс.Карты / 2ГИС

### 7. 🗂️ Мини-CRM (трекер заказов)
- Стадии: Лид → Замер → Проект → Договор → Производство → Монтаж → Отзыв
- AI двигает клиента по стадиям, напоминает, генерирует контент на каждом этапе

### 8. 📊 Аналитик конкурентов (автопилот)
- AI раз в неделю сканирует 5–10 конкурентов
- Находит слепые зоны (метод «Руж»), подсказывает темы

### 9. 🎨 AI-визуализатор (мебель в интерьере клиента)
- Клиент скидывает фото пустой кухни → AI генерирует визуализацию с мебелью
- Лёгкая точка входа вместо полного 3D-проекта

### 10. Аналитика и реклама (V3+)
- Яндекс.Метрика, Яндекс.Директ
- В перспективе: агент, который сам настраивает рекламу

### 11. 💰 AI-сметчик (автоматическая смета)
- Александр вводит размеры + материалы → AI формирует **готовую смету в PDF**
- Позиции, количество, цены, итог — можно сразу отправить клиенту
- Интегрируется с AI-менеджером: агент собрал бриф → автосмета

### 12. 🎬 AI-режиссёр коротких видео
- Александр снимает 3–5 клипов по 10 сек на телефон
- AI даёт сценарий: «Сними руку, которая открывает ящик с доводчиком → крупный план петли → лицо довольного клиента»
- Из клипов → Reels / Shorts / Stories
- **Результат**: профессиональные видео без видеографа

### 13. 📦 Мониторинг цен на материалы
- AI мониторит цены на ЛДСП, МДФ, фурнитуру у поставщиков
- Уведомляет: «Цена на ЛДСП Egger упала на 8% у поставщика X»
- Помогает точнее считать сметы и держать актуальные цены
- **Без автозакупа** — только информирование, решение принимает Александр

### 14. 📱 Сторис-генератор
- AI создаёт серию из 5–7 сторис по одному фото:
  1. «Было» (фото до)
  2. «Процесс» (монтаж)
  3. «Стало» (результат)
  4. «Отзыв клиента»
  5. «Записаться на замер» (CTA)
- Готовый шаблон для Instagram Stories / VK Stories

### 15. 🔁 Авто-ресайкл контента
- Старые посты **не умирают!**
- Через 30–60 дней AI берёт удачный пост и переделывает:
  - Пост → карточка-инфографика
  - Пост → Reels-сценарий
  - Пост → пин Pinterest с SEO-описанием
  - Пост → статья для Дзен
- Один контент живёт **4–5 жизней** на разных площадках
