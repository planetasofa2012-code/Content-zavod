## 1. Анализ текущих площадок

### Instagram (@kuhnya54)
- Аккаунт существует, но не ведётся активно
- **Требования для автопостинга:**
  - Перевести в **бизнес-аккаунт** (или аккаунт автора)
  - Привязать к **странице Facebook/Meta**
  - Instagram Graph API — официальный способ публикации
  - Поддерживает: фото, видео, карусели, Reels, Stories
  - Лимит: до 50–100 публикаций через API за 24 часа
  - Формат фото: только JPEG

### Telegram (@kuhnya154)
- Канал «Kuhnya_54», описание: «Мебель на заказ», 3 подписчика
- Несколько постов опубликовано
- **Автопостинг через Telegram Bot API:**
  - Самый простой из трёх площадок
  - Бот добавляется администратором канала
  - Публикует текст, фото, видео, документы
  - Нет лимитов на количество постов
  - Python-библиотека: `aiogram` или `python-telegram-bot`

### ВКонтакте (группа «Разговоры на кухне»)
- Группа не ведётся активно
- Личная страница: Самосудов Александр
- **Автопостинг через VK API:**
  - Библиотека: `vk_api` (Python)
  - Метод `wall.post` для публикации на стену группы
  - Поддерживает: текст, фото, видео, опросы
  - Нужен API-токен группы (через VK Apps → Manage)

### Pinterest
- Не используется — огромный потенциал для мебельного бизнеса
- Pinterest = **визуальная поисковая система** (не соцсеть!)
- Пины живут **месяцами и годами** (в отличие от постов в соцсетях)
- Люди ищут в Pinterest: «кухня на заказ», «шкаф-купе идеи», «дизайн гардеробной»
- **Автопостинг через Pinterest API v5:**
  - REST API, OAuth 2.0 аутентификация
  - Python-библиотеки: `python-pinterest`, `py3-pinterest`, или через `requests`
  - Создание пинов: POST-запрос с URL изображения, заголовком, описанием, ссылкой
  - Поддержка досок (boards) — можно создать доски: Кухни, Шкафы, Гардеробные и т.д.
  - **Планирование** публикации через параметр `scheduled_at`
  - API открыт для всех разработчиков (нет минимальных требований)

## 2. Анализ конкурентов (методология «Руж»)

> **Принцип**: не копировать — впитывать суть, находить слепые зоны, создавать свой гибрид.

### 5 шагов анализа

| Шаг | Что делать | Для мебельного бизнеса |
|-----|-----------|----------------------|
| **1. Просканировать ДНК** | Изучить сайты, соцсети, рекламу, отзывы конкурентов. Какую «суперсилу» они показывают? | Найти 5–10 мебельных мастерских в городе. Что у них в Insta/VK? Как ведут соцсети? |
| **2. Впитать логику** | Понять операционную систему, не просто креатив. Найти людей, которые строили эту систему | Как конкуренты получают заявки? Какая воронка работает? Откуда трафик? |
| **3. Забрать лучшее** | Не копировать всё — найти самую прибыльную воронку. Какая последовательность даёт 80% дохода? | Какие посты дают больше всего лайков/заявок? Какие фото работают лучше? |
| **4. Найти слепые зоны** | Ахиллесова пята конкурентов: боль клиента, о которой они молчат; формат, который игнорируют | Может, никто не делает честных видео с производства? Никто не показывает провалы? |
| **5. Создать гибрид** | Взять системность одного + скорость другого + глубину третьего → через свою призму | AI-автоматизация (чего нет у других) + живой контент + простая коммуникация |

## 3. Анализ конкурентных решений (сервисы автопостинга)

| Сервис | Площадки | Бесплатно | Особенности |
|--------|----------|-----------|-------------|
| **SMMplanner** | Insta, VK, TG, FB, OK | 50–100 постов/мес бесплатно | Редактор фото, аналитика |
| **LiveDune** | Insta, VK, TG | 7 дней бесплатно | Работает через офиц. API |
| **InSMM** | Insta, VK, TG | 50 постов + 5 stories/мес | Простой интерфейс |
| **Postmypost** | Insta, VK, TG, FB | 14 дней бесплатно | Кросспостинг |
| **NovaPress** | Insta, VK, TG | 10 дней бесплатно | Планировщик |
| **Botmother** | TG, VK, MAX | Есть бесплатный тариф | Конструктор ботов |

### Вывод по сервисам
- Сервисы решают **публикацию**, но НЕ **генерацию контента**
- За генерацию текстов и идей — отвечает AI (Claude/Gemini через OpenRouter)
- **Оптимальный подход**: AI генерирует контент → своя система публикует (без платного сервиса)

## 4. AI для генерации контента

### Что может делать AI (Claude/Gemini)
- ✅ Генерировать посты по фото (Vision API — распознать мебель на фото)
- ✅ Писать тексты разного формата (советы, кейсы, описания)
- ✅ Адаптировать текст под площадку (короче для TG, длиннее для VK, с хештегами для Insta)
- ✅ Генерировать контент-план на неделю/месяц
- ✅ Подбирать хештеги
- ✅ Создавать вовлекающие вопросы и опросы
- ✅ Расшифровывать голосовые сообщения (Whisper) → генерировать пост

### Модели
| Модель | Для чего | Стоимость |
|--------|----------|-----------|
| **Claude Opus** | Лучшие тексты, сложные кейсы | ~$15/1M токенов |
| **Gemini 2.0 Flash** | Быстрая генерация, достаточно для постов | Бесплатно / дёшево |
| **Claude Haiku** | Быстрая адаптация текстов | ~$0.25/1M токенов |

**Рекомендация**: Gemini Flash для ежедневных постов, Claude для важных кейсов.

## 5. Архитектурные варианты

### Вариант А: Telegram-бот (MVP) ⭐ Рекомендован
```
Владелец → Telegram-бот → AI генерация → Публикация (TG + VK + Insta)
```
- Быстро собрать
- Привычный интерфейс (уже есть опыт с ботами)
- Минимум инфраструктуры

### Вариант Б: Веб-панель (Next.js)
```
Владелец → Веб-интерфейс → AI генерация → Публикация
```
- Красивый дашборд
- Контент-календарь
- Сложнее в разработке

### Вариант В: Готовый сервис + AI
```
Владелец → SMMplanner/LiveDune → Ручная вставка AI-текстов
```
- Не нужно писать код публикации
- Но нет автоматизации генерации
- Ежемесячная плата

**Решение**: Начинаем с **Варианта А** (Telegram-бот) — быстрый MVP, потом можно добавить веб-панель.

## 7. Шкала теплоты каналов (Forbes Marketing Guide)

> **Принцип**: идеальная стратегия сочетает каналы разной теплоты и разные воронки.
> Шкала: ❄️ 0 (холодный) → 🔥 10 (горячий, готов к покупке)

### Горячие каналы (7–10/10) — приоритет

| Канал | Теплота | Для мебели | Статус |
|-------|---------|-----------|--------|
| **Реферальная программа** (сарафан) | 🔥 9/10 | Уже работает! Основной источник клиентов | ✅ Есть |
| **CRM / чат-боты** | 🔥 9/10 | AI-агент «Алина» — тёплый контакт с клиентом | ✅ Есть |
| **Avito Ads** | 🔥 7/10 | Аудитория в «покупательском» настроении | 🟡 Нужно |
| **Контекстная реклама** (Яндекс) | 🔥 7/10 | «Кухня на заказ Новосибирск» — горячий запрос | 🟢 V3 |
| **SEO** (Pinterest, Дзен) | 🔥 7/10 | Долгосрочный бесплатный трафик | 🟡 V2 |

### Тёплые каналы (4–6/10) — контент-завод

| Канал | Теплота | Для мебели | Статус |
|-------|---------|-----------|--------|
| **Бесплатные продукты** (чек-листы, калькулятор) | 🌤️ 5/10 | Касание 1 воронки: «7 ошибок при заказе кухни» | 🟡 MVP |
| **SMM** (соцсети) | 🌤️ 4/10 | Контент-завод! Лайки (2/10), комменты (5/10), личные сообщения (8/10) | 🔴 MVP |

### Холодные каналы (1–3/10) — не приоритет

| Канал | Теплота | Комментарий |
|-------|---------|------------|
| Таргетированная реклама | ❄️ 4/10 | Можно позже для усиления |
| Посевы (seeding) | ❄️ 3/10 | «Сарафанный эффект», но этические вопросы |
| Холодные продажи | ❄️ 1/10 | **Продажа в лоб умерла** — не использовать |
| Баннерная реклама | ❄️ 1/10 | Низкий CTR, «баннерная слепота» |

### Вывод по каналам для мебели

```
ГОРЯЧИЕ (конверсия):   Сарафан ✅ + AI-агент ✅ + Авито Ads 🟡
ТЁПЛЫЕ (прогрев):      Контент-завод (SMM) 🔴 + бесплатные продукты 🟡
ДОЛГОСРОЧНЫЕ (SEO):    Pinterest + Дзен + лендинг 🟡
```

**Ключевой инсайт**: Ваш сарафан (9/10) + AI-агент (9/10) — это уже два самых горячих канала. Контент-завод добавляет прогрев (4–5/10), а Pinterest/Дзен/Авито дают SEO-трафик (7/10). Вместе — полная воронка!

## 8. Заметки и выводы

1. **Самый простой путь** — Telegram-бот как интерфейс управления + публикация на 7+ площадок
2. **Instagram** требует бизнес-аккаунт + страницу Facebook — нужно настроить
3. **VK API** — самый простой из соцсетей для автопостинга
4. **Pinterest** — лучшая площадка для мебели! Пины живут годами, SEO-трафик
5. **Авито** — горячий канал (7/10), аудитория в покупательском настроении
6. **AI генерация** — основная ценность проекта (не просто кросспостинг, а умная генерация)
7. **AI-агент на каждой площадке** — консультация → расчёт → запись на замер
8. **Воронка «3 касания»** — бесплатная польза → прогрев → лёгкий вход (не продажа в лоб!)
9. **«Парадокс Барсика»** — AI говорит на языке клиента, просто и по-человечески
10. **Лендинг** — центральный хаб, 11 блоков, цель — заявка на бесплатный замер
11. **Фото** — ключевой контент. Бот должен уметь работать с фото
12. **Голосовые** — Whisper для расшифровки → AI для генерации поста. Минимум усилий
