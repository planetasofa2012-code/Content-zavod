# 🏭 Контент-завод

> AI-система для мебельного бизнеса: генерация контента, публикация на 7+ площадок, консультация клиентов 24/7.

**Автор:** Александр Самосудов — мебель на заказ в Новосибирске (кухни, шкафы, гардеробные).

---

## Что это

Один человек ведёт весь бизнес. На маркетинг и контент времени нет.
«Контент-завод» решает это:

- Сфотографировал готовую работу → отправил боту → посты сами ушли на 4+ площадки
- Клиент написал в группу → AI-агент ответил, рассчитал цену, записал на замер
- Утром пришёл вопрос от AI-интервьюера → ответил голосовым → стало постом и базой знаний

## Компоненты

| # | Компонент | Что делает |
|---|-----------|-----------|
| 1 | **Контент-бот** | Принимает фото + описание → генерирует посты → публикует |

| 3 | **AI-интервьюер** | Задаёт вопросы «как клиент» → ответы = база знаний + посты |
| 4 | **Лендинг** | Сайт-визитка: портфолио, калькулятор, форма заявки |

## Быстрый старт

### 1. Настройка

```bash
# Клонировать / перейти в папку проекта
cd "content zavod"

# Создать виртуальное окружение
python -m venv venv
venv\Scripts\activate        # Windows
# source venv/bin/activate   # Linux/Mac

# Установить зависимости
pip install -r requirements.txt

# Скопировать и заполнить конфиг
cp .env.example .env
```

### 2. Заполнить `.env`

```env
TG_CONTENT_BOT_TOKEN=   # токен бота-контент-завода (от @BotFather)
TG_AGENT_BOT_TOKEN=     # токен бота для клиентов
TG_CHANNEL_ID=          # ID канала @kuhnya154 (начинается с -100...)
OWNER_ID=               # твой Telegram ID (узнать: @userinfobot)
OPENROUTER_API_KEY=     # ключ с openrouter.ai
```

### 3. Запуск

```bash
python -m bot.main
```

Ожидаемый вывод:
```
🏭 Контент-завод: запуск...
✅ Контент-бот запущен
✅ AI-агент запущен
```

## Как пользоваться

### Создать пост

1. Открой **контент-бот** в Telegram
2. Отправь фото готовой мебели
3. Добавь описание: `«Кухня МДФ эмаль, скандинавский стиль, фурнитура Blum»`
4. Выбери тип поста: Портфолио / Кейс / Совет / Отзыв
5. Проверь предпросмотр → нажми ✅

Посты уйдут в Telegram-канал и VK-группу автоматически.

### AI-интервьюер

1. В контент-боте нажми «🎙️ Интервью»
2. Выбери «Случайный вопрос»
3. Ответь текстом или голосовым
4. Ответ превратится в пост и сохранится в базу знаний
