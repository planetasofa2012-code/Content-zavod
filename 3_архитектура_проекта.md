## 4. База данных (Supabase)

### 4.1. Схема таблиц

#### Таблица `posts` — опубликованные посты
```sql
CREATE TABLE posts (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    created_at TIMESTAMP DEFAULT NOW(),
    
    -- Контент
    text_original TEXT,          -- Исходный текст от Александра
    photo_url TEXT,              -- URL фото в хранилище
    voice_transcript TEXT,       -- Расшифровка голосового (если было)
    
    -- Сгенерированный контент
    text_telegram TEXT,          -- Текст для Telegram
    text_vk TEXT,                -- Текст для VK
    text_instagram TEXT,         -- Текст для Instagram
    text_pinterest TEXT,         -- Текст для Pinterest
    text_avito TEXT,             -- Текст для Авито
    
    -- Метаданные
    post_type VARCHAR(50),       -- portfolio / case / tip / review / story
    funnel_stage VARCHAR(20),    -- attract / warmup / convert (касание 1/2/3)
    furniture_type VARCHAR(50),  -- kitchen / wardrobe / closet / shelf
    hashtags JSONB,              -- Хештеги по площадкам
    
    -- Статус публикации
    published_telegram BOOLEAN DEFAULT FALSE,
    published_vk BOOLEAN DEFAULT FALSE,
    published_instagram BOOLEAN DEFAULT FALSE,
    published_pinterest BOOLEAN DEFAULT FALSE,
    published_avito BOOLEAN DEFAULT FALSE,
```

## 9. Принципы архитектуры

| Принцип | Как реализуем |
|---------|-------------|
| **Модульность** | Каждый компонент — отдельная папка. Можно включать/выключать модули |
| **Единая точка AI** | Все модули используют один `ai/openrouter.py` — легко сменить модель |
| **Единая БД** | Все данные в Supabase — нет дублирования |
| **Publisher-паттерн** | Все площадки реализуют `base.Publisher` — легко добавить новую |
| **Agent-паттерн** | Все агенты наследуют `base_agent.py` — единая логика, разные API |
| **Промпты отдельно** | Все промпты в `ai/prompts/` — легко редактировать без кода |
| **FSM** | Состояния диалога через aiogram FSM — предсказуемое поведение |
| **Fail-safe** | Ошибки не роняют бота — логирование + уведомление Александру |
